Описать шаблон класса для хранения данных. 

Данные должны хранится, например, в виде массива.  

Шаблон должен включать в себя функции 
+ добавления данных,
+удаления данных,
+ вывода данных на консоль ( info() ) 

+ и метод, возвращающий минимальный элемент массива. 

Продемонстрировать работу шаблона на объектах как встроенного типа, 

int 
float
char
bool

так и собственного класса. Для этого создать собственный класс и перегрузить в нём нужные функции-операции.
